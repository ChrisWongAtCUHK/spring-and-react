# README
## Reference
https://github.com/kantega/react-and-spring

## How to run

```
mvn clean install
java -jar target/spring-and-react-0.0.1-SNAPSHOT.jar
```

## Dev
```
mvn spring-boot:run
```
In frontend/ folder
```
npm run start
```

## Docker
```
docker build -t springbootapp.
docker run -p 8080:8080 -t springbootapp
```
